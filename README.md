# 🥪 weather-app

# 📝 Description
This repository contains the weather app project. The project uses webpack. The [WeatherAPI](https://www.weatherapi.com/) API is used to retrieve the weather.

## 🎨 Styling
Used normalize.css

## ✨ Features

- Using Flexbox
- Using Grid
- Using Animate.css
